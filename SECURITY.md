# Security Policy

AegisFlow is built for hostile internet exposure. See
[docs/SECURITY.md](docs/SECURITY.md) for the full posture and
[docs/THREAT_MODEL.md](docs/THREAT_MODEL.md) for the threat model.

## Reporting a vulnerability

Email **security@aegisflow.example** with details and a PoC if available.
Please do not open public issues for vulnerabilities. We aim to acknowledge
within 2 business days and to remediate Critical issues within 7 days.

## Supported versions

The `main` branch receives security fixes. Tagged releases are supported for 12
months from release.
